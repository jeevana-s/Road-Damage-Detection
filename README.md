# 🚧 NHAI AI Monitoring System

A smart web-based platform that uses AI concepts to monitor national highways in real time. It helps in identifying traffic conditions, detecting accidents, and managing alerts efficiently through an interactive dashboard.

---

## 🚀 Key Features

* 📊 **Real-Time Dashboard**

  * View live data such as accidents, traffic levels, alert counts, and active cameras

* 🚨 **Alert Monitoring**

  * Track alerts categorized by severity levels like High, Medium, and Low

* 📈 **Data Analytics & Reports**

  * Visualize traffic patterns and accident statistics using charts

* 🤖 **Detection Module**

  * Upload images to analyze and identify traffic or accident scenarios

* 🗺️ **Highway Overview**

  * Monitor different highway zones (North, South, East, West)

* ☎️ **Emergency Support**

  * Quick access to control room contact for immediate response

---

## 🛠️ Technologies Used

### Frontend

* React with TypeScript
* Tailwind CSS for styling
* Chart.js for data visualization
* Lucide React for icons

### Backend

* Flask (Python framework)
* RESTful API architecture
* Cross-Origin Resource Sharing (CORS) enabled
